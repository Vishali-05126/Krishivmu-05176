# **App Name**: Landing Site Builder

## Core Features:

- HTML scaffolding: Quickly generates index.html with a basic HTML structure and placeholders for content. 
- CSS Generation: Generates a basic style.css with essential styles and placeholders for further customization.
- Preview and Editing: Allow users to see a preview of their landing page design.
- Code Customization: Ability to download the source HTML and CSS for local code changes and customizations. This tool utilizes the AI to help reason what components depend on each other to simplify further editing by the developer.
- Deployment: Simplified process to publish landing page (index.html and style.css) directly.

## Style Guidelines:

- Primary color: Soft Lavender (#D0BFFF) for a calm and approachable feel.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Muted Purple (#A483B9) to complement the primary, drawing visual attention to specific areas or elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, neutral and objective feel.
- Use simple, geometric icons related to landing page creation (e.g., a pen, a cursor, a file).
- Clean, modern layout with plenty of white space. Focus on readability.
- Subtle animations (e.g., fade-ins) on page load.
